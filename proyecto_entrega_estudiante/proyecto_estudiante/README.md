# Tarea - Sistema de Estudiantes (hecho como estudiante)

Contenido:
- backend/: API en Spring Boot (simple, en memoria)
- frontend/: html/js para consumir la API
- docs/: diagramas plantuml y arquitectura
- tests: ejemplos con JUnit4 + Mockito

Instrucciones:
1. Backend: ir a backend/ y correr `mvn spring-boot:run` (Java 17).
2. Frontend: abrir frontend/index.html (o servirlo con nginx).
3. Tests: `mvn test` dentro de backend/.
